# FIS Live Scoring - Prettier Results

A userscript that cleans up FIS live-scoring results for TV display.

## Install on Chrome

1. Install the [Tampermonkey](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) extension.
2. Open [`fis-results-prettier.user.js`](./fis-results-prettier.user.js) and click **Raw**.
3. Tampermonkey will prompt you to install — confirm.
4. Visit any page on `https://fis.live-scoring.com/` to see the script run.
